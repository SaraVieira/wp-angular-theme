Wp-Angular

Write responsive wordpress templates in a declaretive way and load content asynchronous.

This Wordpress plugin is currently work in progress!

Introduction

This plugin extends the populair Wordpress content management system (CMS) with the power of AngularJS. The web is emerging and responsivenes of your website becomes more impotant. Different screen resolutions and limitations in bandwith provides a challenge to give the end user the best experience of the content. Designers and developers have to adapt there ways of designing and building websites to this new world. Wp-Angular can help you to adapt your worpress website to the new world.

Currently Wordpress does not optimal support the demand for responsive websites. Most of the magic curently happens serverside. To make responsive websites more logic on the client side is required. Angular provides a MVW framwork writen in javascript wich allows you to build well structured responsive websites.

By integrating Angular and Wordpress you can gain from the rich CMS functionality of wordpress and the felxibility of Angular to build responsive themes in a declaretive way. AngularJS provides the possibility to load content asynchronous which makes your website more responsive and gives a better user experience.
