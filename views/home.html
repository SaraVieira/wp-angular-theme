Write wordpress templates in an declaretive and asounchonus way.

Intorduction

This plugin combines the populair Wordpress content management system (CMS) with the power of AngularJS. The web is emerging with the introduction of the smartphone more the responsivenes of website has become more important. Responsivnes in screen sizes but also in bandwith.

Currently Wordpress does not optimal support the demand for responsiveness because all the content is renderd on the server. Angular provides us a MVW framwork wich allows you to build well structured javascript appliction.

By intergrating Angular into Wordpress you can gain from the rich CMS functionality of wordpress and the felxibility of angular to build responsive templates in pure javascript.
