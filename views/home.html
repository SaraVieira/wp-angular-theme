Introduction

This plugin extends the populair Wordpress content management system (CMS) with the power of AngularJS. The web is emerging and responsivenes of your website becomes more and more impotant. With the introduction of the smartphone different screen sizes and limitations in bandwith became a chalange. Designers and developers have to adapt there way of designing and building websites to this new enviroment. By making websites more responsive logic will move from the server to the client wich is now posible because of the browser wich become more mature.

Currently Wordpress does not optimal support the demand for responsiveness website. Most of the magic will happen server site in wordpress. Angular provides us a MVW framwork writen in javascript wich allows you to build well structured applictions client side.

By intergrating Angular into Wordpress you can gain from the rich CMS functionality of wordpress and the felxibility of Angular to build responsive templates in pure javascript way.
