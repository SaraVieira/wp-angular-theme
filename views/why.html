Why use Wp-Angular

Wp-Angular allows you to write your Wordpress themes in an declarative way. Instaed of mixingup php and html server side you do everytihng in html client side. The following examples will give the same result only one is in the traditional php + html style the other one is done in the Angular way.

Both example show a list of the 5 most recent post. The first example is done in Wordpress style where php and Html is mixed to generat the list. The second example is done in het Angular way wich is a pure html implementation.

Wordpress style

<?php 
	$args = array( 'numberposts' => '5' ); 
	$recent_posts = wp_get_recent_posts( $args ); 
	foreach( $recent_posts as $recent ){ 
		echo '<li>';
		echo '<a href="' . get_permalink($recent["ID"]) . '">';
		echo $recent["post_title"];
		echo '</a>';
		echo '</li>';
	} 
?>

AngularJs way

<ul wp-posts numberofposts="5"> 
	<li wp-post> 
		<a ng-href="{{url}}" >{{title}}</a> 
	</li> 
</ul>
