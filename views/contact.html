About this project

This project started out of a frustration that Wordpress templates are moste of the time very messy and unstructured. When you download a third part template it is very hard to customize it to your own needs. In my opinion Wordpress is one of the best opensource content management systems. It is very easy to learn for you customers and it has enough capebilities to work for small and medium size companies where multiple content managers have to work together.

This porject is hosted and maintaind in Github. It is still under development and never tested on load and performace. The project exists of two sub-projects wp-angular-plugin an wp-angular-theme. The plugin contains the AngularJs integration and a set op directives. The theme contains examples and this documentation. If you would like to contribute please contact me.
