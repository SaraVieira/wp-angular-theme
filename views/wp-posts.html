Wp-Posts

This directive is an container for loading multiple posts and paginate trough them. To display the posts the wp-post directive is used. For all the availible attributes have a look at this page.

This directive gives you the option to order and filter on fields

Post can be orderde on the following fields by making use of the order and orderby attributes. For more information see Order & Orderby Parameters on the wordpress website.

  • order
    • ASC
    • DESC
  • orderby
    • none
    • ID
    • author
    • title
    • name
    • date
    • modified
    • parent
    • rand
    • comment_count
    • comment_count
    • meta_value
    • meta_value_num
    • post__in

The amount of posts which should be load can be set with the 'numberofposts' attibute. See example 3 for an implementation
