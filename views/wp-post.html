Wp-Post

The wp-post directives loads the content of one item asynchronous base on the id of the post

After de content is loaded the all the attributes of the post will be availibe on the scope of the directive. Here is an list of all the availible variable

  • ID
  • title
  • status
  • type
  • author
  • content
  • parent
  • link
  • date
  • modified
  • format
  • slug
  • guid
  • excerpt
  • menu_order
  • comment_status
  • ping_status
  • sticky
  • date_tz
  • date_gmt
  • modified_tz
  • modified_gmt
  • password
  • post_meta
  • meta
  • featured_image
  • terms

{{title}}
